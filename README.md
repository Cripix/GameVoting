# GameVoting Development
Simple voting plugin Sourcemod.net for TF2, CSS, CSGO etc

It's necessary: 
Add into addons\sourcemod\configs\databases.cfg
```
"gamevoting"
{
		"driver"    "sqlite"
		"database"  "gamevoting-local"
}
```
